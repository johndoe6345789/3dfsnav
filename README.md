# Jurassic UNIX Navigator (pyglet)

Cinematic homage to the Jurassic Park "UNIX system" moment, rendered with pyglet.

## Install
```bash
python -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
```

## Run
```bash
python run_app.py
```

## Controls
- Mouse drag: rotate
- Mouse wheel: zoom
- Click: open dir / open file
- Backspace: go up
- Enter: open selected (hover)
- Escape: quit

## Tests
```bash
python run_tests.py
```
